My research is in Natural Language Processing and Machine Learning. I am interested in grammatical induction and parsing methods for syntactic-semantic analysis and translation of natural languages.

In 2005 I completed my PhD at the Universitat Politècnica de Catalunya (UPC) under the supervision of Lluís Màrquez. From 2006 to 2009 I was a postdoc at MIT/CSAIL with Michael Collins. From 2009 to 2014 I was a Ramón y Cajal researcher at the TALP Research Centre of UPC.

Since June 2014 I am senior researcher at XRCE.

Software

  • egstra: Exponentiated-Gradient algorithms for STRuctured prediction. This C++ code implements EG algorithms to train max-margin and log-linear models for structured prediction tasks. The package includes other popular learning algorithms as well, namely Perceptron and SGD. The current package includes algorithms for first-order dependency parsing.
  • boostree : A C++ implementation of Schapire and Singer's AdaBoost.MH, with decision-tree weak learning. The algorithm is suited for multi-label classification problems, and sparse example representations based on binary-valued features (typical for NLP problems). This code was used to develop many CoNLL shared task systems which obtained state-of-the-art-results. Check the documentation of the package.
